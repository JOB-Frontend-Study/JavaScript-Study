## 2. 다음 중 false 값이 아닌 것은?

① undefined

② NaN

③ null

④ []

⑤ -0

<br/>
<details>
<summary>정답</summary>
<pre>
④번.
false로 취급되는 값은 다음과 같다.
- false
- undefined
- null
- 0, -0
- NaN
- ‘’(빈 문자열) 
→ *[](빈 배열)은 true

</details>
